# ID Document Fraud Detection System

This project is a prototype AI system that analyzes uploaded ID document images and generates a fraud risk report.

The system uses Computer Vision and OCR to detect potential tampering in ID documents.

## Features

* Upload ID document image
* Image validation
* Tampering detection
* OCR text extraction
* Fraud risk analysis

## Technologies Used

* Python
* OpenCV
* Tesseract OCR
* Streamlit
* NumPy

## How to Run

Install dependencies:

pip install -r requirements.txt

Run the application:

streamlit run app.py
