# ID Document Fraud Detection System

This project is a prototype AI system that analyzes an uploaded ID document image and generates a fraud risk report.

## Features

- Upload ID document image
- Image validation
- Tampering detection
- OCR text extraction

## Technologies Used

- Python
- OpenCV
- Tesseract OCR
- Streamlit

## How to Run

Install dependencies:

pip install -r requirements.txt

Run the application:

streamlit run app.py

## Project Pipeline

Upload Image
→ Image Validation
→ Tampering Detection
→ OCR Extraction
→ Fraud Detection Report
